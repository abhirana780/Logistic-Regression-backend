# Logistic Regression Intelligence Engine (Backend) 🧠

This is the power source for the AI Strategy Suite. It handle dynamic dataset generation, logistic regression model training, and real-time probability calculations.

## 🚀 Key Features

- **Multi-Domain Data Generator:** Generates unique, realistic datasets for HR, Marketing, and Finance.
- **AI Training API:** Exposes endpoints for training Scikit-Learn Logistic Regression models on-the-fly.
- **Mathematical Decoder:** Provides raw coefficients and intercepts for educational transparency.
- **Flask Framework:** A lightweight, high-performance Python backend.

## 🛠️ Setup Instructions

1. Navigate to the `/backend` directory.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the engine:
   ```bash
   python app.py
   ```
   The API is live at `https://logistic-regression-backend.onrender.com`.

## 🧪 Tech Stack

- Python
- Flask
- Scikit-Learn
- Pandas
- NumPy
